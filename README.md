## Syfte
Att träna på FetchAPI anrop tillsammans med async/await och try/catch och träna på att filtrera, sortera data.

## Uppgifter
Uppgiften består av flera delar som framgår i listan nedan:
 * Läsa in från API och presentera i tabell-format,
 * Sortera data i bokstavsordning från rubrik på respektive kolumn,
 * Filtrera tabell-data utifrån sökfras.
